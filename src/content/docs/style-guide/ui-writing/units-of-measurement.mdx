## Basics

-   Separate units of measurement from their value with a single space.
-   For measurements over time, insert a forward slash between the unit of measurement and the time unit.

| Do        | Don't    |
| --------- | -------- |
| 29.5K rpm | 29.5Krpm |
| 6.33 ppm  | 6.33ppm  |
| 16 KB/s   | 16 KB s  |
